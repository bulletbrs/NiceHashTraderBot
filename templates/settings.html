How to Get API Credentials
  1. Log in to your NiceHash account
  2. Go to Settings > API Keys
  3. Create a new API key with appropriate permissions:
    • View account balance
    • Manage hashpower orders
    • View hashpower order history
  4. Copy the Organization ID, API Key, and API Secret
  5. Enter these values in the form above
Security Notice: Keep your API Secret secure and never share it with anyone. If you suspect your credentials have been compromised, immediately revoke them from your NiceHash account.
